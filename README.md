# Overview

The Student Management System (SMS) frontend is a web-based application that allows users to manage student records efficiently. It provides features such as adding, updating, and deleting student details, along with viewing a list of registered students. The frontend is built using modern web technologies to ensure a smooth and responsive user experience.
